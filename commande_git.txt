configuration initial:
dans la cli
on de place dans le dossier qui va acceuilli notre projet git
git config --global user.name "nom_du_compte_github"
git config --global user.mail @email
git config --global color.diff auto
git config --global color.status auto
git config --global color.branch auto
git config --global core.edit vim
git config --global merge.tool vimdiff
pour initialser un depot :git init ///
git remote add commande-sql https://github.com/junior-st/commande-sql.git
//établissons la connexion ssh avec le depot distant
 ssh-keygen -t ed25519 -C "chakoutiojunior@gmail.com" //generation de notre cle privee/public
//on copie la cle public dans le presse papier
clip <~/.ssh/id_ed25519.pub (sur Windows )
-relier le depot distant au depot locale vite fait: git remote add ===lienssh_du_depot===
git branch -M main
git push -u origin main
En indiquant  origin  suivi du nom de la branche, ici  main , toutes les modifications faites sur la branche  main  seront bien envoyées sur cette branche et ne mettront pas à jour tout votre repository.
